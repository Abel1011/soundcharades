# 🎧 SoundCharades

**Can you guess what you're hearing?** SoundCharades is a multiplayer-ready trivia game where AI-generated sounds and music are the only clues. Listen carefully, pick the right answer, and climb the leaderboard.

Built for the **ElevenLabs + turbopuffer Hackathon**.

## 🎮 How It Works

1. **Pick a category** — Movies, Series, Videogames, Countries, Food, Music, or create your own custom theme.
2. **Listen** — Each round plays AI-generated sound effects or an original music composition that represents a concept.
3. **Guess** — Choose from 4 options. The faster you answer, the more points you earn.
4. **Use hints wisely** — Text hints and voice-over narration are available, but each one costs you points.
5. **See your results** — Get a full breakdown with accuracy, streaks, and a letter grade.

### Round Types

| Type | What you hear | How it's made |
|---|---|---|
| **Sound Chain** | 4 layered ambient loops + 2 spoken phrases | ElevenLabs Sound Effects API |
| **Riddle Song** | An original 30-second composition with cryptic lyrics | ElevenLabs Music API (via Gemini composition plan) |

### 🧠 Difficulty System — Powered by turbopuffer

The difficulty system is the core innovation of SoundCharades and it's **entirely powered by turbopuffer's vector search**.

Instead of manually tagging concepts as "easy" or "hard", we leverage **semantic vector proximity** to dynamically control difficulty:

- Every concept is stored in turbopuffer as a **768-dimensional embedding** (generated by Gemini `text-embedding-005`).
- When building a round, we query turbopuffer with **cosine distance** to find the correct answer's nearest neighbors.
- **Hard mode** picks distractors that are semantically *close* to the correct answer — e.g., if the answer is "The Matrix", the options might be "Blade Runner", "Inception", and "Ghost in the Shell".
- **Easy mode** picks distractors that are semantically *far apart* — making it obvious which one the sounds represent.
- **Medium** blends both strategies for a balanced challenge.

This approach means difficulty **scales automatically** as more concepts are added — no manual curation needed. turbopuffer's **sub-millisecond query latency** makes this feasible in real time: each game session performs multiple vector queries to assemble rounds, and the entire session is built in under a second.

> **Why turbopuffer?** We needed a vector database that could handle real-time gameplay queries with minimal latency. turbopuffer's speed allows us to run multiple cosine similarity searches per session without any perceivable delay, making the vector-based difficulty system feel instant to players. It also powers the custom quiz feature — once new concepts are upserted, they're immediately queryable for building rounds.

### How Concepts Are Created

The seed pipeline (and custom quiz generator) works in stages:

1. **Gemini Pro** generates concepts with rich metadata: name, description, SFX prompts, music composition plan.
2. **Gemini text-embedding-005** creates a 768-dimensional vector for each concept.
3. **ElevenLabs Sound Effects** generates 4 ambient loops (5s each) + 2 spoken phrases (4s each).
4. **ElevenLabs Music** composes a 30-second original song based on a Gemini-written composition plan with cryptic lyrics.
5. **ffmpeg** normalizes all audio to consistent loudness (`loudnorm I=-14:TP=-1:LRA=11`).
6. **turbopuffer** stores the concept with its embedding for vector-based retrieval.

### Custom Quiz Generation

Players can type any theme (e.g. "90s punk rock bands") and the app generates a full quiz on the fly:

- A **polling-based** architecture handles the 2–4 minute generation time (POST creates a job, GET polls every 2s).
- Progress is tracked through stages without revealing concept names (no spoilers!).
- Generated concepts are stored in turbopuffer with `category: "custom"` and are **instantly queryable** — thanks to turbopuffer's speed, players can start a round with their freshly generated concepts immediately after creation.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- ffmpeg installed and in PATH
- API keys for: ElevenLabs, Google Gemini, turbopuffer

### Setup

```bash
# Clone the repo
git clone https://github.com/your-username/soundcharades.git
cd soundcharades

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Fill in your API keys:
#   ELEVENLABS_API_KEY=...
#   GEMINI_API_KEY=...
#   TURBOPUFFER_API_KEY=...
```

### Seed the Database

Before playing, you need concepts in turbopuffer:

```bash
# Seed 4 concepts per category (default)
npm run seed

# Clean and re-seed from scratch
npm run seed:clean

# Auto-seed (picks the smallest category, good for cron jobs)
npm run seed:auto
```

### Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start playing!

## 🧩 Key Features

- **AI-Generated Audio** — Every sound effect and music track is unique, created by ElevenLabs specifically for each concept.
- **Vector-Based Difficulty** — turbopuffer's cosine similarity search dynamically controls difficulty by measuring semantic proximity between concepts. No manual tagging, no arbitrary labels — just math.
- **Custom Themes** — Generate a quiz from any topic in minutes using the full AI pipeline.
- **SoundBuddy** — An animated mascot that reacts to your gameplay with mood expressions (happy, excited, thinking, sad, cool).
- **Smart Scoring** — Base points + speed bonus, with penalties for replays, hints, and voice-over assistance.
- **Game History** — Track your progress across sessions with stats and letter grades.
- **Auto-Seed** — Keep the database growing with a cron-friendly script that balances categories.
